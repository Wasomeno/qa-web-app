Create `handlers/test_scenario_v2.go` with all CRUD endpoints:
- GET /test-scenarios/:id (V2 format with sections/stats)
- PATCH /test-scenarios/:id (update title/description/status)
- PATCH /test-scenarios/:id/sections/:sectionId/test-cases/:tcId (update test case)
- PATCH /test-scenarios/:id/sections/:sectionId/test-cases/reorder (reorder test cases)
- POST /test-scenarios/:id/sections/:sectionId/test-cases (add test case)
- DELETE /test-scenarios/:id/sections/:sectionId/test-cases/:tcId (delete test case)
- POST /test-scenarios/:id/generate-test-case (generate automation for single test case)
- POST /test-scenarios/:id/generate-section (generate automation for all test cases in section)
